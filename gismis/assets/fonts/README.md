# Font Assets

This directory should contain the following font files:

## NotoSerifSC (Title Font)
- NotoSerifSC-Regular.ttf
- NotoSerifSC-Medium.ttf
- NotoSerifSC-SemiBold.ttf
- NotoSerifSC-Bold.ttf

## NotoSansSC (Body Font)
- NotoSansSC-Regular.ttf
- NotoSansSC-Medium.ttf
- NotoSansSC-SemiBold.ttf
- NotoSansSC-Bold.ttf

Download from Google Fonts:
- https://fonts.google.com/noto/specimen/Noto+Serif+SC
- https://fonts.google.com/noto/specimen/Noto+Sans+SC
